Boys vs Supes — Battle Sim

An unofficial, lovingly stupid FastAPI fan project that pits Vought supes against The Boys and rolls dice until somebody’s head explodes.

View on GitHub How it works

15 characters

Homelander, Stormfront, Soldier Boy, Butcher, Kimiko, MM, Frenchie and friends — each with stats and powers.

Battle simulator

Pick up to 7 per side, POST to /battle, get a winner, MVP, full round-by-round log.

FastAPI + React

Typed Pydantic models, OpenAPI docs at /docs, React+Vite frontend talking to the API.

Live demo

Same battle logic as the Python backend, ported to JS so it runs right here in your browser.

Try it locally

git clone https://github.com/anton-cheg/the-boys-vought-api
cd the-boys-vought-api/backend
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload

# in another terminal
cd ../frontend
npm install && npm run dev

Then open http://localhost:5173 for the arena, or http://localhost:8000/docs for the OpenAPI playground.

Sample request

POST /battle
{
  "side_a": ["homelander", "stormfront"],
  "side_b": ["butcher", "kimiko", "soldier-boy"],
